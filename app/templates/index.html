  • Cel aplikacji

    Aplikacja została stworzona w celu utrwalenia wiedzy zdobytej podczas zajęć projektowych z kursu programowania w języku Python oraz przedmiotu zaawansowane technologie webowe. Jej celem było praktyczne zastosowanie poznanych technologii do stworzenia funkcjonalnej aplikacji webowej.

    Opis działania aplikacji

    Aplikacja umożliwia użytkownikom pobieranie opinii o produkcie z Ceneo.pl, pobieranie plików z opiniami w formatach CSV, JSON, XLSX. Została zaprojektowana z naciskiem na prostotę użytkowania, bezpieczeństwo oraz przejrzystość interfejsu.

  • Zastosowane biblioteki

    W projekcie wykorzystano m.in. biblioteki takie jak Flask,
    BeautifulSoup, numpy, pandas, requests.

    Zastosowane podejścia

    W projekcie zastosowano podejście oparte na podziale logiki aplikacji na funkcje i moduły. Do pobierania danych wykorzystano technikę web scrapingu. Dane są analizowane i przekształcane z użyciem bibliotek do analizy danych. Projekt korzysta z systemu kontroli wersji Git oraz struktury folderów ułatwiającej zarządzanie danymi i kodem. Aplikacja została wykonana z użyciem szablonów HTML oraz podstaw routingu we Flasku.
